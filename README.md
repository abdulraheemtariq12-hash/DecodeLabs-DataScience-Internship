# Iris Classification ML Project

## 📌 Description
This is a beginner Machine Learning project using the Iris dataset.  
The goal is to classify flower species using Logistic Regression.

## 📊 Dataset
I used the built-in Iris dataset from scikit-learn.

## ⚙️ Steps Performed
- Loaded dataset using sklearn
- Converted to Pandas DataFrame
- Checked missing values
- Removed duplicates
- Performed data analysis
- Split data into train and test sets
- Trained Logistic Regression model
- Evaluated model performance

## 🧠 Model Used
- Logistic Regression (scikit-learn)

## 🚀 How to Run
```bash
pip install scikit-learn pandas
python main.py
```
